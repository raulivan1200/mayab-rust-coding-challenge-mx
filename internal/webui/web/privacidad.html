Política de privacidad

Última actualización: 12 de julio de 2026

1. Responsable y alcance

Esta política explica cómo Raúl Iván Rivera Pérez, operador de Mayab Arbitraje BTC (la "Aplicación"), trata información cuando usas el bot de Discord o el dashboard público.

2. Información procesada

Cuando ejecutas un comando, Discord envía a nuestro endpoint los datos necesarios de la interacción, que pueden incluir identificadores de Discord, servidor o canal, permisos, nombre del comando, opciones y el texto que escribas. La Aplicación procesa esos datos para autenticar la solicitud y responder.

El dashboard no usa cookies ni crea identificadores persistentes de visitantes. Puede guardar localmente en tu navegador preferencias visuales, modo de depuración y, si lo configuras manualmente, el token de administración.

3. Finalidades y base de uso

Procesamos la información para ejecutar comandos, mostrar el estado simulado, responder preguntas, prevenir abuso, diagnosticar fallos y mantener la seguridad y disponibilidad de la Aplicación. El tratamiento ocurre porque solicitas el servicio al interactuar con el bot y por el interés legítimo de operar una demostración segura.

4. Proveedores y transferencias

La Aplicación utiliza Discord para recibir y responder interacciones y Google Cloud Run para alojar el servicio. Si usas el comando de IA, el texto de tu pregunta y el contexto necesario de la simulación pueden enviarse a la API de NVIDIA para generar la respuesta. Estos proveedores pueden procesar datos conforme a sus propias políticas y desde otros países.

5. Conservación

La Aplicación no crea perfiles de usuarios ni una base de datos destinada a conservar el contenido de comandos de Discord. Las solicitudes se procesan en memoria. Los proveedores de infraestructura pueden conservar temporalmente registros técnicos conforme a su configuración y políticas. La información local del dashboard permanece en tu navegador hasta que la elimines.

6. Divulgación

No vendemos información personal. Podemos divulgar información si es necesario para cumplir una obligación legal, responder a una autoridad competente, proteger derechos o seguridad, investigar abuso o mantener el servicio con los proveedores descritos.

7. Seguridad

Las interacciones se verifican mediante la firma criptográfica de Discord antes de procesarse. Usamos transporte HTTPS y controles razonables de acceso. Ningún sistema es completamente infalible, por lo que no podemos garantizar seguridad absoluta.

8. Menores

La Aplicación no está dirigida deliberadamente a menores de la edad mínima exigida por Discord en su país. Si crees que se trató información de un menor de forma indebida, contáctanos para revisarla.

9. Tus opciones y derechos

Puedes dejar de usar o desinstalar la Aplicación en cualquier momento y borrar los datos locales desde la configuración de tu navegador. Para solicitar acceso, corrección, eliminación u oponerte a un tratamiento aplicable, contacta al desarrollador mediante el perfil o la ficha de esta aplicación en Discord. Podremos pedir información suficiente para verificar y localizar la solicitud.

10. Cambios

Podemos actualizar esta política cuando cambien la Aplicación, sus proveedores o las obligaciones aplicables. Publicaremos la versión vigente en esta misma URL con su fecha de actualización.
